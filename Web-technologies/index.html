WEB TECHNOLOGIES

HTML

HTML (HypterText Markup Language) is the fundamental building block of the Web and defines the content and structure of a web page. Below are some common page elements that HTML is used for:

  • Headings
  • Paragraphs
  • Images
  • Links
    • Lists

Learn more about HTML with the w3Schools HTML Totorial..

CSS

CSS CSS (Cascading Style Sheets) is used to describe a web page's appearance and presentation. Below are some common uses of CSS:

  • Fonts
  • Colors
  • Layout
  • Backrounds

Learn more about HTML with the w3Schools CSS Totorial.

JAVASCRIPT

JavaScript is the programming language of the web and controls the functionality and behaviour of a web page. Below are some common uses of JavaScript:

  • Add Interactivity to a Web Page
  • Create Web Apps
  • Create Web Games & Animations

A great JavaScript referance is The Modern JavaScript Tutorial.
